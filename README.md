# :rocket: | Desafio Dio: Calculadora Rankeada
O projeto consiste em calcular o ranking do jogador de acordo com a quantidade de vitórias e derrotas. 

# ❓ | O que é praticado?

- Variáveis
- Operadores
- Laços de Repetição
- Estruturas de Decisões
- Funções

## 🎯 | Objetivo Principal

Criar uma **função** que recebe como **parâmetro** a quantidade de vitórias e derrotas de um jogador, retornar o resultado para uma **variável** e realizar o cálculo para atribuir um _ranking_.

## 🏆 | Rankings do Jogo

| Condição de Vitórias        | Patente   |
|-----------------------------|-----------|
| Menor que 10                | Ferro     |
| Entre 11 e 20               | Bronze    |
| Entre 21 e 50               | Prata     |
| Entre 51 e 80               | Ouro      |
| Entre 81 e 90               | Diamante  |
| Entre 91 e 100              | Lendário  |
| Maior ou igual a 101        | Imortal   |
